# Contributing to CrystalAtlas

Bug reports, feature requests and pull requests are welcome.

When reporting a filename-parsing issue, include anonymised example filenames and the regular expression being used. Do not upload confidential experimental images unless you have permission to share them.

For code changes:

1. Fork the repository.
2. Create a focused branch.
3. Test the application from source.
4. Open a pull request describing the change and how it was tested.
